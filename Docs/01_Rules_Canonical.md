# 01 Rules Canonical

本文档是游戏规则的唯一正式来源。术语解释见 `Docs/02_Rules_Glossary.md`，数据结构见 `Docs/05_Data_Schema.md`，测试用例见 `Docs/07_Test_Cases.md`，决策与未解决问题见 `Docs/08_Decision_Log.md`。

## 1. 游戏目标

本游戏是足球主题双人卡牌对战。比赛结束时，进球数更多的一方获胜。

如果比赛结束时双方进球数相同，允许平局。MVP 阶段不强制加时、点球大战或重赛。

## 2. 牌组与手牌

- 每名玩家使用 20 张球员卡。
- 整副牌组就等于手牌，没有牌库、抽卡、洗牌概念。
- 比赛开始时，20 张球员卡全部作为手牌。
- 双方可以看到互相的手牌情况。
- 同一玩家的 20 张球员卡无重复。
- 每名玩家必须且只能有 1 名门将。

## 3. 球员卡基础数据

球员卡包含基础资料、稀有度、位置类型、属性和技能。

### 3.1 基础资料

以下字段属于球员卡基础资料，仅用于卡牌展示，MVP 阶段不参与任何战斗公式、定位球公式、门将公式、体力公式、稀有度积分或回收概率。

- `HeightCm`：身高，单位厘米，建议类型 `int32`。
- `WeightKg`：体重，单位千克，建议类型 `int32`。
- `BirthDate`：出生日期，建议格式 `YYYY-MM-DD`。

### 3.2 稀有度

球员卡稀有度使用英文枚举名，中文作为显示名。

| 枚举名 | 中文显示 | 积分 |
| --- | --- | ---: |
| `WorldClass` | 世界级 | 7 |
| `Continental` | 洲际级 | 5 |
| `National` | 国家级 | 3 |
| `Regional` | 地区级 | 2 |
| `Common` | 普通级 | 1 |

### 3.3 初始牌组稀有度积分

初始牌组稀有度积分定义为：比赛开始时，玩家 20 张球员卡的稀有度积分总和。

规则：

1. 只在比赛开始时计算。
2. 比赛过程中，手牌、已消耗区、弃牌区变化，不会重新计算该积分。
3. 该积分用于进攻次数加成判定。
4. 该积分用于双方进攻次数相同时的初始先后手判定。
5. 该积分不等于当前手牌实时积分。
6. 正式规则术语只使用“初始牌组稀有度积分”。

### 3.4 位置类型

位置类型枚举：

- `A`：进攻。
- `M`：组织。
- `D`：防守。
- `GK`：门将。

非门将球员可以包含多种位置类型，例如 `A/M` 或 `M/D`。

门将只能是 `GK` 类型，不允许叠加其他位置类型，例如 `GK/A`、`GK/M`、`GK/D` 都不合法。

### 3.5 基础属性

通用属性数值区间为 1-6：

- 射门
- 盘带
- 传球
- 跑位
- 盯人
- 抢断
- 速度
- 强壮
- 体力
- 远射

门将属性数值区间为 1-6：

- 手控球
- 站位
- 反应
- 高空球
- 预判
- 一对一

规则文本中的旧称“身体”统一为“强壮”。

### 3.6 球员技能

球员卡可以配置最多 3 个进攻技能。

技能是否可用由技能配置中的触发行动点范围决定。技能配置表中可以存在多个同名技能，例如多个“传中”，但触发行动点范围可以不同。

`SkillDefinition` 是完整技能配置数据源。球员卡正面只显示技能名称 `SkillDisplayName` 和可触发行动点范围 `TriggerActionPointRange`；`SkillDefinition` 中的其他字段用于规则计算，不显示在卡牌正面。

规则计算字段可用于技能详情页、tooltip、战斗日志或调试信息，但球员卡正面不显示角色要求、跑位区域要求、结算步骤、进球归属角色、球员耗费规则、公式引用或完整结算公式。

## 4. 区域与槽位

### 4.1 场地槽位

场地槽位为双方共用。双方共用槽位用于形成抢空间玩法。

底层逻辑只需要理解以下概念：

- 中场区域
- 前场区域
- 后场区域
- 当前进攻方
- 当前防守方

中线左右只影响玩家视角和画面表现，不作为底层规则条件。

### 4.2 放置限制

- 进攻方 `D` 类型不可放前场。
- 防守方 `A` 类型不可放后场。
- 多类型球员只要存在合法位置类型，即可按合法类型放置。例如 `A/M` 放在前场可视为进攻型合法放置。
- 每个卡槽只能放 1 张卡。

### 4.3 战术球员

部署位置与自身位置类型匹配的球员算作战术球员：

- `A` 类型放置在前场。
- `M` 类型放置在中场。
- `D` 类型放置在后场。

多类型球员只要有一个类型符合部署区域，即算战术球员。

终结公式中，战术球员人数优势提供加成：

- 多出至少 2 名战术球员：点数 +1。
- 多出至少 3 名战术球员：点数 +2。
- 人数优势加成最高 +2。

### 4.4 攻防球员

参与当前公式结算的球员称为攻防球员。常见角色包括：

- 持球球员
- 盯人球员
- 跑位球员
- 协防球员
- 门将

攻防球员不等于所有已部署球员，只有被选入当前公式的球员才参与结算。

### 4.5 已消耗区

该区域正式命名为“已消耗区”，英文建议为 `Consumed Zone`。

已消耗区是本场比赛中已经被使用、但尚未进入弃牌区的球员卡临时区域。进入已消耗区的球员未来可以根据回收规则返回手牌。

区别：

- 已消耗区：未来可能返回手牌。
- 弃牌区：本场比赛不会返回手牌。

运动战中打到攻防区并结算后的球员进入已消耗区。定位球战术中被耗费的球员也进入已消耗区。

基础回收原则：已消耗区球员按照体力相关规则返回手牌。具体概率公式仍记录在 `Docs/08_Decision_Log.md` 的未解决问题中。

## 5. 进攻次数与先后手

### 5.1 进攻次数计算

每名玩家基础进攻次数为 3 次。

然后比较双方初始牌组稀有度积分：

- 初始牌组稀有度积分较高的一方，额外获得 1 次进攻次数。
- 如果双方初始牌组稀有度积分相同，则双方都不获得该项额外进攻次数。

之后每名玩家进行附加随机次数判定：

- 掷 D6。
- 1-3：额外获得 1 次。
- 4-6：额外获得 2 次。

### 5.2 初始先后手判定

当比赛开始时，双方总进攻次数相同，需要确定初始先后手：

1. 先比较双方初始牌组稀有度积分。
2. 初始牌组稀有度积分较高的一方后手。
3. 如果初始牌组稀有度积分也相同，则双方掷 D6。
4. 点数大的一方后手。
5. 如果掷点相同，则继续掷 D6，直到双方点数不同。

该判定只用于比赛开始时确定初始先后手。比赛过程中不再反复比较初始牌组稀有度积分。

### 5.3 进攻顺序队列

比赛开始时根据双方总进攻次数和初始先后手生成进攻顺序队列。比赛过程中按照已生成的进攻顺序队列执行。

进攻次数较高的一方先进攻，然后双方交替进攻。当次数较少的一方只剩 1 次进攻时，次数较多的一方连续进攻，直到双方都剩 1 次；随后次数较少的一方先执行最后 1 次，次数较多的一方再执行最后 1 次。

示例：甲方进攻次数 5 次，乙方进攻次数 3 次。

1. 甲方进攻 1 次。
2. 乙方进攻 1 次。
3. 双方交替进行，直到甲方剩余 3 次，乙方剩余 1 次。
4. 甲方连续进攻 2 次，使甲方也剩余 1 次。
5. 乙方进攻最后 1 次。
6. 甲方进攻最后 1 次。
7. 双方进攻次数都为 0，比赛结束。

## 6. 掷点与公式通用规则

### 6.1 掷点类型

- 普通掷点：D6。
- 行动点：D12。

### 6.2 比较点数

比较点数就是掷点结果点数。

当公式要求双方都获取比较点数时：

1. 进攻方先掷 D6，得到进攻方比较点数。
2. 防守方后掷 D6，得到防守方比较点数。
3. 双方比较点数彼此独立。
4. 掷点顺序需要写入 MatchLog。

### 6.3 小数规则

所有公式计算结果允许保留一位小数。

- 公式中出现 `/2` 时，结果保留一位小数。
- 门将属性一半计入时，结果保留一位小数。
- 公式比较时直接比较一位小数结果。
- UI 展示时最多显示一位小数。
- 不进行向上取整、向下取整或四舍五入到整数。

### 6.4 点数快速压制

点数快速压制只适用于双方都需要获取比较点数的公式。

规则：

1. 双方都掷 D6 获取比较点数时，才可能触发点数快速压制。
2. 如果一方掷出 6，另一方掷出 1 或 2，则掷出 6 的一方直接赢得该次判定。
3. 不适用于只由一方掷点的判定公式。
4. 不适用于行动点 D12。

### 6.5 公式类型

过渡公式：

- 进攻方胜利：进入下一结算步骤。
- 防守方胜利：当前进攻回合结束。
- 过渡公式胜利本身不直接产生进球。

判定公式：

- 根据具体技能或定位球表处理。
- 判定公式不一定比较双方属性。

终结公式：

- 进攻方胜利：进球。
- 防守方胜利：不进球，当前进攻回合结束。
- 平局：按平局判定规则处理。

### 6.6 当前进攻回合结束

以下情况都会结束当前进攻回合：

- 终结公式产生进球。
- 终结公式防守方胜利。
- 过渡公式防守方胜利。
- 判定公式结果为射偏、越位、失败或其他结束结果。
- 进攻方无合法球员。
- 进攻方无可用技能。
- 规则明确写明“本回合结束”的其他情况。

## 7. 平局判定

### 7.1 单人公式平局

公式点数相同时，比较攻防球员体力。

- 体力较高的一方获胜。
- 如果体力相同，防守方获胜。
- 如果门将参与了该公式，则防守方获胜。

### 7.2 多人公式平局

当公式中包含多名攻防球员时：

- 进攻方体力值取参与公式的进攻方球员体力之和。
- 防守方体力值取参与公式的防守方球员体力之和。
- 体力总和较高的一方获胜。
- 如果体力总和相同，防守方获胜。
- 如果门将参与了该公式，则防守方获胜。

### 7.3 无协防球员

如果防守方不选择协防球员：

- 协防球员提供的属性值视为 0。
- 协防球员体力视为 0。

## 8. 行动点状态机

每次进攻回合开始时掷 D12，得到行动点。

- 行动点 1：本方进攻结束，并执行球员罚下判定。
- 行动点 2-8：运动战，进入部署阶段和技能结算。
- 行动点 9-12：定位球，跳过常规部署，进入同一套定位球结算表。行动点 9、10、11、12 本身没有差异。

行动点 1 的罚下随机范围是手牌。被罚下球员进入弃牌区。

## 9. 部署阶段与技能选择

### 9.1 部署阶段

行动点为 2-8 时，进入运动战部署阶段。

双方按进攻方、防守方顺序交替操作。每次操作时，玩家可以：

1. 打出一张合法球员卡进入攻防区。
2. 点击部署完毕。

当一方点击部署完毕后，该方本回合不再继续部署。

当系统检测一方没有合法可打出的球员时，可以自动执行与点击部署完毕相同的处理流程。

### 9.2 部署完毕后的技能选择

双方都部署完毕后，进攻方只能选择当前行动点匹配的球员卡及其技能。

具体规则：

1. 当前行动点为 2-8 时，进入运动战技能选择。
2. 进攻方从已部署的本方球员中选择可作为持球球员的球员。
3. 该球员必须配置有至少一个技能，且该技能的触发行动点范围包含当前回合行动点。
4. 如果没有任何合法球员或合法技能，则当前进攻回合结束，不进球。
5. 技能配置表中可以存在多个同名技能，例如多个“传中”，但触发点数范围可以不同。
6. 技能是否可用由技能配置中的触发点数范围决定。

### 9.3 无合法球员处理

玩家点击部署完毕与系统检测无合法球员，应进入同一套后续判定流程。

判定规则：

1. 如果进攻方没有合法球员或没有合法技能，则当前进攻回合结束，不进球。
2. 如果防守方没有合法球员，则进攻方获得系统进球。
3. 系统进球不归属于具体球员。
4. 系统进球可以在日志中记录为 `SystemGoal`。
5. 如果双方都没有合法球员，则优先视为进攻方手牌不足，当前进攻回合结束，不进球。

## 10. 门将规则

### 10.1 门将数量与类型

每名玩家的 20 张球员卡中，必须有且只有 1 名门将。

门将只能是 `GK` 类型。门将不允许叠加其他位置类型，例如 `GK/A`、`GK/M`、`GK/D` 都不合法。

不存在多名门将共享一次发动机会的情况。

### 10.2 发动门将

防守方在运动战中可以发动门将。

规则：

1. 每名玩家单场比赛只能发动门将一次。
2. 发动门将后，门将进入对应防守区域参与终结公式。
3. 门将相关属性的一半计入运动战终结公式。
4. 门将属性一半保留一位小数。
5. 发动后记录为已使用状态。
6. 发动后只记录为已使用状态。

运动战门将属性对应：

- 远射：站位。
- 内切射门：手控球。
- 传中低球：反应。
- 传中高球：高空球。
- 直塞：一对一。
- 传控：手控球。

### 10.3 定位球中的门将

定位球不使用“发动门将”概念。定位球结算中按定位球表直接引用门将属性。

## 11. 手牌不足与三抽一

### 11.1 三抽一候选球员不足

三抽一时，双方最多各提供 3 张候选球员。

如果一方实际可提供的候选球员少于 3 张，仍按实际可提供数量参与。

比较双方实际提供的候选卡数量差：

- 少 1 张：少的一方本次比较点数 -2。
- 少 2 张：少的一方本次比较点数 -4。
- 少 3 张：视为手牌不足。

### 11.2 候选球员 0 张处理

只要一方实际可提供候选球员数量为 0 张，即视为该方手牌不足。

如果双方候选球员数量都是 0 张，则视为进攻方手牌不足。

## 12. 通用技能结算表

### 12.1 远射（单人）

进攻方可选择：

A. 直接射门：终结公式。

- 持球球员远射 + 进攻方比较点数 VS 盯人球员抢断 + 防守方比较点数 + 2。
- 进攻方比较点数为 1-2 时，直接射偏并结束当前进攻回合。

B. 直射死角：判定公式。

- 进攻方掷 D6 两次，总和为 11 或 12 则直接进球。
- 其余结果为射偏，当前进攻回合结束。

进球队员：持球球员。

### 12.2 内切射门（单人）

进攻方可选择：

A. 直接射门：终结公式。

- (持球球员射门 + 持球球员盘带) / 2 + 进攻方比较点数 VS 盯人球员抢断 + 防守方比较点数 + 2。
- 进攻方比较点数为 1-2 时，直接射偏并结束当前进攻回合。

B. 直射死角：判定公式。

- 进攻方掷 D6 两次，总和为 11 或 12 则直接进球。
- 其余结果为射偏，当前进攻回合结束。

进球队员：持球球员。

### 12.3 传中（双人）

跑位球员必须为进攻方前场球员。

进攻方选择传球方式为高球或低球：

- 选择高球：进攻方掷 D6，1-4 为高球，5-6 为低球。
- 选择低球：进攻方掷 D6，1-4 为低球，5-6 为高球。

高球：终结公式。

- (持球球员传球 + 跑位球员强壮) / 2 + 进攻方比较点数 VS (盯人球员抢断 + 协防球员强壮) / 2 + 防守方比较点数 + 2。

低球：终结公式。

- (持球球员传球 + 跑位球员射门) / 2 + 进攻方比较点数 VS (盯人球员抢断 + 协防球员盯人) / 2 + 防守方比较点数 + 2。

进球队员：跑位球员。

### 12.4 直塞（双人）

跑位球员必须为进攻方前场球员。

进攻方掷 D6 判定传球方式：

- 1-2：脚下球。
- 3-4：身后球。
- 5-6：反越位。

脚下球：终结公式。

- (持球球员传球 + 跑位球员跑位) / 2 + 进攻方比较点数 VS (盯人球员抢断 + 协防球员盯人) / 2 + 防守方比较点数 + 2。

身后球：

1. P1 过渡公式：(持球球员传球 + 跑位球员速度) / 2 + 进攻方比较点数 VS (盯人球员盯人 + 协防球员速度) / 2 + 防守方比较点数 + 1。
2. 进攻方比较点数为 1-2 时，传出底线，当前进攻回合结束。
3. P1 进攻方胜利后，执行 P2 判定公式：防守方掷 D6，4-6 造越位成功，当前进攻回合结束；1-3 进攻方跑位球员单刀，进入单刀判定。

反越位：判定公式。

- 进攻方掷 D6，6 时跑位球员单刀，进入单刀判定。
- 1-5 时越位，当前进攻回合结束。

进球队员：跑位球员。

### 12.5 传控（双人）

跑位球员必须为进攻方中场球员。

进攻方掷 D6 判定推进方式：

- 1-2：传球推进。
- 3-4：盘带推进。
- 5-6：跑动推进。

传球推进：终结公式。

- (持球球员传球 + 跑位球员传球) / 2 + 进攻方比较点数 VS (盯人球员抢断 + 协防球员盯人) / 2 + 防守方比较点数 + 2。

盘带推进：终结公式。

- (持球球员盘带 + 跑位球员传球) / 2 + 进攻方比较点数 VS (盯人球员抢断 + 协防球员盯人) / 2 + 防守方比较点数 + 2。

跑动推进：终结公式。

- (持球球员跑位 + 跑位球员盘带) / 2 + 进攻方比较点数 VS (盯人球员盯人 + 协防球员盯人) / 2 + 防守方比较点数 + 2。

进球队员：跑位球员。

## 13. 定位球结算表

行动点 9-12 时进入定位球。行动点本身没有差异，进入定位球后额外掷 D6 判定定位球类型：

- 1-2：角球。
- 3-4：远距离任意球。
- 5：近距离任意球。
- 6：点球。

### 13.1 角球

双方最多各准备 3 名手牌球员进行三抽一，得出跑位球员和协防球员。

进攻方选择传球方式为高球或低球：

- 选择高球：进攻方掷 D6，1-4 为高球，5-6 为低球。
- 选择低球：进攻方掷 D6，1-4 为低球，5-6 为高球。

高球：终结公式。

- 跑位球员强壮 + 进攻方比较点数 VS (协防球员强壮 + 门将高空球) / 2 + 防守方比较点数 + 2。

低球：终结公式。

- 跑位球员射门 + 进攻方比较点数 VS (协防球员盯人 + 门将反应) / 2 + 防守方比较点数 + 2。

球员耗费：跑位球员及协防球员进入已消耗区。

### 13.2 远距离任意球

进攻方任选一名手牌球员作为持球球员。

进攻方可选择：

A. 直接射门：终结公式。

- 持球球员远射 + 进攻方比较点数 VS 门将站位 + 防守方比较点数 + 2。
- 进攻方比较点数为 1-2 时，直接射偏并结束当前进攻回合。

B. 重炮轰门：判定公式。

- 进攻方掷 D6 两次，两次总和大于等于 11 即进球，否则当前进攻回合结束。

球员耗费：持球球员进入已消耗区。

### 13.3 近距离任意球

进攻方任选一名手牌球员作为持球球员。

进攻方可选择：

A. 直接射门：终结公式。

- max(持球球员射门, 持球球员传球) + 进攻方比较点数 VS 门将手控球 + 防守方比较点数 + 1。

B. 角度射门：判定公式。

- 持球球员射门与传球属性值和达到 8 时才可选择此项。
- 进攻方掷 D6 两次，两次总和大于等于 9 即进球，否则当前进攻回合结束。

球员耗费：持球球员进入已消耗区。

### 13.4 点球

进攻方任选一名手牌球员作为持球球员。

进攻方可选择：

A. 直接射门：终结公式。

- max(持球球员射门, 持球球员传球) + 进攻方比较点数 VS 门将预判 + 防守方比较点数 - 3。

B. 勺子点球：判定公式。

- 进攻方掷 D6，2-6 进球，1 射失并结束当前进攻回合。

球员耗费：持球球员进入已消耗区。

## 14. 其他结算

### 14.1 单刀

进攻方可选择：

A. 直接射门：终结公式。

- 单刀球员射门 + 进攻方比较点数 + 1 VS 门将一对一 + 防守方比较点数。

B. 挑射：判定公式。

- 进攻方掷 D6，4-6 挑进；1-3 挑射失败，当前进攻回合结束。

### 14.2 补射

补射为终结公式：

- 补射球员射门 + 进攻方比较点数 VS 对方门将预判 + 3。

### 14.3 打空门

打空门为判定公式：

- 射门方掷 D6，2-6 进球，1 射飞并结束当前进攻回合。
